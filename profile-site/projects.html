Localization Mission Coordination on ROS and Gazebo environment

Dec 2020 — Jan 2021

Implemented a geometric method to localize 8 flags position using 3 robots in an unknown environment using Python and ROS. The solution coordinates agents, processes sensor observations, and computes flag positions robustly in the presence of noise.

Visual-Inertial Odometry for Agent’s Motion

Nov 2020 — Jan 2021

  • Implemented the Structure-from-Motion (SfM) algorithm on camera frames pipeline to obtain scaled pose.
  • Implemented quaternion integration algorithm for IMU (gyroscope) readings to obtain Euler angles and fuse orientation with visual estimates.

Autonomous UAV for Medical Rescue Applications

Sep 2019 — Jul 2020

This project is the first prototype of a quad-plane designed and manufactured at Cairo University. The target of the project was to design, manufacture and implement an autonomous quad-plane to be used in medical rescue applications.

The vehicle is composed of multiple systems divided into modules: air-frame design and manufacturing, control system design and implementation, and path planning & image processing. My role focused on control design, system integration and autonomous behaviors.
